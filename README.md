# Olympic Games Historical Analytics Using SQL & Power BI

## Project Overview

This project analyzes more than a century of Olympic Games history using SQL, Excel, and Power BI. The objective was to uncover insights related to athlete participation, medal distribution, country performance, sports evolution, host city trends, and Olympic growth through data-driven analysis and interactive dashboards.

The project combines complex SQL querying, data modeling, exploratory data analysis (EDA), and Power BI visualization to answer real-world analytical questions from historical Olympic datasets.

---

## Objectives

- Analyze Olympic Games participation trends over time.
- Study medal distribution across countries and sports.
- Identify dominant nations and sports.
- Explore gender participation and event distribution.
- Examine athlete demographics and performance patterns.
- Develop interactive dashboards for data-driven exploration.

## Key Insights

### Olympic Growth
- Summer Olympics have hosted more games than Winter Olympics.
- The number of sports and events has steadily increased over time.

### Participation Trends
- Athlete participation has grown significantly across Olympic editions.
- Male participation remains higher than female participation, though the gap has narrowed in recent decades.

### Medal Distribution
- The United States consistently leads the overall medal tally.
- Countries such as Germany, Russia, the United Kingdom, and China have demonstrated strong historical performance.

### Sport Dominance
- Athletics contains the highest number of Olympic events.
- Ice Hockey and Hockey show strong regional dominance from specific countries.

### Country Performance
- USA remains the most successful Olympic nation historically.
- Several countries demonstrate high performance efficiency despite smaller athlete delegations.
